Timetable Generator

Converts the school's web timetable to a .timetable file for the Class Timetable App.




Why is there a password?

Because I don't want everybody on the Internet to be able to download our timetables. That's also why the school name isn't mentioned. Ask the person who gave you this link for the password. If you can't figure it out email me using your school email.
