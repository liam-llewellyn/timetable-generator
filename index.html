.timetable Generator for NBSC Manly Campus

Converts the school's web timetable to a .timetable file for the Class Timetable App.

WARNING: Putting your school password into random websites is a very dangerous thing to do. I know nothing about password security, and AI was used in the creation of this website. The source code is on GitHub, but I wouldn't trust this website. However, I promise this code will only use your password to log into the school's website, and will not be stored. There is no central server, all code runs on your machine.



Advanced

Instructions

  • Download the Class Timetable App.
  • Download your timetable with the form above.
  • Within Class Timetable, click Settings, then Timetable Tools, then Import Timetable and choose the file downloaded before.
  • Change the colours to your liking.
